▼ 💡 UML Notation Tips & Guide
🔑 Access Modifiers 
+  → public          -  → private
#  → protected       ~  → package-private
None → default access
📝 Attribute Format 
access name: Type
Example: - name: String
Example: + age: int
⚙️ Method Format 
access name(params): ReturnType
Example: + getName(): String
Example: + setName(name: String): void
🔧 Params with types 
+ setData(name: String, id: int): void
🏗️ Constructor 
Use class name as method name:
Example: + Student(name: String)
Note: Default constructor is auto-generated!
📌 Static members 
Use {static} in curly braces
Example: - count: int {static}
Example: + getInstance(): MyClass {static}
🔒 Final fields 
Use {final} in curly braces
Example: - PI: double {final}
🔗 Combined modifiers 
Use {static, final} with comma
Example: - MAX: int {static, final}
📐 Abstract methods 
Use {abstract} in curly braces
Example: + draw(): void {abstract}
🔹 Abstract class 
Write in Class Name box:
  <<abstract>> ClassName
🟡 Interface 
Write in Class Name box:
  <<interface>> InterfaceName
🟠 Enum 
Write in Class Name box:
  <<enum>> EnumName
🔗 Multiple Classes / Relations 
Write extends or implements in Class Name box:
Example: class Car extends Vehicle
Example: class Dog implements Animal
